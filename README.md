# 访问历史标记 Chrome插件

## 简介
访问历史标记是一个Chrome插件，可自动为网页中已访问的链接添加可视化标记，通过颜色区分访问时间，帮助用户快速识别浏览轨迹。


## 主要功能

### 1. 智能链接标记
- **时间梯度色标：**
  - 最近访问（1小时内）：🔴 红色标记
  - 今日访问：🟠 橙色标记
  - 更早访问：🟢 绿色标记
- **样式自定义：** 支持边框、背景高亮、下划线三种标记形式，可自定义各时间段的颜色

### 2. 隐私保护
- 支持"仅标记当前会话"模式
- 可设置自动清理周期（1天/1周/1月）
- 不记录无痕模式下的访问历史

### 3. 页面适配
- 自动处理动态加载内容，如无限滚动页面
- 通过MutationObserver监听新出现的链接，实时标记

### 4. 用户交互
- **右键菜单功能：**
  - 强制标记/忽略特定链接
  - 临时禁用特定页面的标记功能
- **快捷键支持：**
  - Alt+H 显示/隐藏所有标记
  - Alt+J 手动添加标记
- **悬浮控制按钮：** 可在页面右下角快速切换标记显示/隐藏

## 安装方法

### 从Chrome网上应用店安装
1. 访问[Chrome网上应用店](https://chrome.google.com/webstore)--未上架
2. 搜索"访问历史标记"
3. 点击"添加到Chrome"

### 手动安装
1. 下载本项目的zip文件并解压
2. 打开Chrome浏览器，在地址栏输入 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择解压后的文件夹

## 使用方法
1. 安装完成后，点击工具栏中的插件图标打开设置面板
2. 在设置面板中可以：
   - 切换启用/禁用状态
   - 选择标记样式（边框/背景高亮/下划线）
   - 自定义颜色设置
   - 更改隐私保护选项
3. 浏览网页时，已访问的链接将自动根据您的设置进行标记
4. 使用Alt+H快捷键可以随时显示/隐藏标记

## 隐私说明
本插件高度重视用户隐私：
- 所有数据均存储在本地，不会上传到云端
- 可选择仅记录当前会话的数据
- 可设置自动清理周期
- 不会记录无痕模式下的访问历史
